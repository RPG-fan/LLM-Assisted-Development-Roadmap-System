# Key Success Milestones

These milestones define the key stages of user engagement and project maturity we aim to achieve:

-   **Minimum Viable Product (MVP) / Minimum Viable Gameplay Loop (MVG):**
    -   User can `[perform core action A, e.g., log in]`.
    -   User can `[perform core action B, e.g., upload a data file]`.
    -   User can `[perform core action C, e.g., see a basic chart]`.
    -   User can `[perform core action D, e.g., save their work]`.
    *Goal: The user understands the basic mechanics and can achieve a primary objective.*

-   **Core Functionality Complete:**
    -   All core components have basic functional implementations.
    -   Users can complete the primary workflow from start to finish without critical failures.
    -   Basic error handling and feedback are present.
    -   A clear progression or value path is available to the user.
    *Goal: The user experiences the primary value proposition and feels a sense of accomplishment or utility.*

-   **Alpha Readiness Criteria:**
    -   All core components are implemented and reasonably stable.
    -   Major bugs and blockers are resolved.
    -   Basic tutorial, tooltips, or onboarding is available.
    -   The product is usable from start to a significant progression point.
    -   Performance is acceptable on target hardware/platforms.
    *Goal: The project is feature-complete enough for initial external testing and feedback, showcasing the intended experience.*

-   **Beta Readiness Criteria:**
    -   All planned features for the 1.0 release are implemented.
    -   Significant content, data, or configuration options are available.
    -   Advanced features (e.g., complex calculations, AI behaviors) are functional.
    -   Data persistence (save/load, database) is robust.
    -   Comprehensive testing (unit, integration, performance, user acceptance) has been conducted.
    -   Balancing/tuning passes completed for key variables.
    *Goal: The project is feature-complete, content-rich, and polished enough for broader testing and near-release quality.*
